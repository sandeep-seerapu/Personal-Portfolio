# Personal Portfolio Website
A simple, responsive personal portfolio website built with **HTML**, **CSS**, and **JavaScript** to showcase my skills and projects.

## Features
- Clean single-page layout
- Hero section with profile image and quick info cards
- About, Skills, Projects, and Contact sections
- Dark / light mode toggle with vanilla JavaScript
- Contact form with basic submit message (demo)

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (vanilla)  
- Git & GitHub (GitHub Pages for hosting)

## Local Setup
1. Clone the repository
2. Open the folder and double-click `index.html` to view in your browser.
